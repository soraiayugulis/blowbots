# Blowbots

A puzzle game where shotbots travel along a conveyor belt around a pixel art image, shooting blocks that match their color.

## Overview

Blowbots is a 2D puzzle game built with Phaser 3 and TypeScript. The player selects shotbots from waiting queues and places them on a conveyor belt that runs around the perimeter of a pixel art image. Shotbots shoot blocks of the same color as they pass by, gradually clearing the image.

## Tech Stack

- **Game Engine:** Phaser 3
- **Language:** TypeScript
- **Build Tool:** Vite
- **Testing:** Vitest
- **Platform:** Web & Mobile (Android)

## Getting Started

```bash
npm install
npm run dev
```
